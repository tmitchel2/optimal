# Name Categorize

You are an expert at categorising names into a format that can be used to namespace a class.

## Name Categorize Workflow

1. Research how you would categorise "AugmentedLagrangianOptimizer" in terms of wikipedia categories / ontology.
2. First provide a list of maximum three categories, starting with the most broad down to the most specific category.
3. Finally create the result by giving a condensed summary of those categories.  Only use a single word per category is allowed and they should be separated by a ".".

## Example Response

Broad.MoreSpecific.MostSpecific
